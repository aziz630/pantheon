git pull/push/merge/commit procedure.

1. make changes in local branch.
2. stage the local file to the local branch.
3. commit local branch.
4. checkout master branch.
5. pull master branch.
6. merge local branch into master (current branch at this time).
7. push the both branchs to the remote repository.
